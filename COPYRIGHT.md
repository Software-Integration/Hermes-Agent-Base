# Copyright and Contribution Boundary

This document explains the copyright boundary for Hermes Agent Base.

## Repository-Controlled Material

Unless a file states otherwise, the following are intended to be repository-controlled material:

- original source files created for this repository
- repository-specific glue code
- configuration authored for this repository
- integration adapters authored for this repository
- tests, documentation, and operational files authored for this repository
- repository-specific modifications made by this repository's contributors

These repository-controlled portions are intended to follow the repository licensing model:

- `AGPL-3.0-only` as the open-source license
- optional separate commercial licensing where granted by the rights holder

## Third-Party Material

Third-party material keeps its own ownership, copyright notices, and license terms.
This includes upstream projects, vendored code, copied license texts, and any separately licensed dependency.

Repository publication does not convert third-party material into property of the repository operator or contributors.

## Upstream Attribution Rule

When a component comes from an upstream project:

- preserve the upstream project name
- preserve the upstream license
- preserve upstream copyright ownership where applicable
- do not imply relicensing of the upstream component unless the upstream license itself allows it and the rights holder has actually done it

## Contributions

By contributing repository-controlled material to this repository, you represent that you have the right to contribute that material.

Unless explicitly agreed otherwise in writing, contributions to repository-controlled material are accepted on the basis that they may be distributed under:

- `AGPL-3.0-only`
- and, where applicable to repository-controlled material, the repository's separate commercial licensing model

This statement applies only to material that contributors actually control the rights to contribute.
It does not override the license of third-party code.

## No Ownership Misstatement

No file in this repository should claim or imply that this repository owns upstream projects such as:

- `NousResearch/hermes-agent`
- `open-policy-agent/opa`
- `qdrant`
- `sentence-transformers`
- `Valkey`
- `gVisor`

Those projects remain owned by their respective upstream rights holders.
